# ZavgarApp — Приложение для завгаров

Управление автопарком: учёт автомобилей, водителей, запчастей и технического обслуживания.

## Функции

- **Автопарк** — учёт автомобилей (марка, модель, госномер, пробег, статус)
- **Водители** — база водителей с контактами и документами
- **Склад запчастей** — приход/расход, остатки, минимальные запасы
- **ТО и обслуживание** — планирование по пробегу, история ремонтов
- **Отчёты** — статистика по автопарку

## Установка

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск
python -m zavgar_app.main
```

Или на Windows:
```
ZAPUSTIT.bat
```

## Требования

- Python 3.10+
- PySide6 >= 6.5.0
- Windows 10/11 (для тёмной темы)

## Архитектура

```
zavgar_app/
├── __init__.py          — версия, метаданные
├── main.py              — точка входа
├── db.py                — SQLite: схема, CRUD, миграции
├── models/              — дата-классы (Vehicle, Driver, Part, ...)
└── ui/
    ├── theme.py         — современная тёмная тема
    └── main_window.py   — главное окно с боковой панелью
```

## Дизайн

Современная тёмная тема:
- Закруглённые кнопки (border-radius 12-16px)
- Тени через QGraphicsDropShadowEffect
- Градиентные фоны (настраиваемые)
- Палитра: индиго (#6366f1), фиолетовый (#8b5cf6)
- Шрифт: Segoe UI
