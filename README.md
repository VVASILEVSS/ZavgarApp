# ZavgarApp

Профессиональное десктопное приложение для управления автопарком, складом запчастей и планированием ТО.

![Python](https://img.shields.io/badge/Python-3.10+-blue.svg)
![PySide6](https://img.shields.io/badge/PySide6-6.5+-green.svg)
![License](https://img.shields.io/badge/License-MIT-yellow.svg)

## Возможности

### 🚗 Автопарк
- Учёт автомобилей (марка, модель, год, госномер, VIN)
- Статусы: активный, на ТО, списан, продан
- Типы: легковой, грузовой, фургон, автобус, спецтехника

### 👤 Водители
- Учёт водителей (ФИО, телефон, права, категория)
- Срок действия водительского удостоверения
- Дата найма и заметки

### 🔧 Склад запчастей
- Каталог запчастей с артикулами и категориями
- Учёт остатков и минимальных запасов
- История прихода/расхода с привязкой к автомобилям

### 🛠️ ТО и обслуживание
- Графики ТО по пробегу (масло, фильтры, тормоза, шины)
- Автоматический расчёт следующего ТО
- История выполненных работ с затратами
- Проверка просроченных ТО

### 📈 Отчёты
- Статистика: авто в парке, водители, запчасти
- Затраты на ТО по автомобилям
- Средняя стоимость обслуживания

### 🎨 Интерфейс
- Современный дизайн (Linear/Vercel inspired)
- Тёмная и светлая темы
- Анимированные кнопки и стеклянные карточки
- Переключение тем одной кнопкой

## Установка

### Требования
- Python 3.10+
- PySide6 6.5+

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск

**Windows:**
```bash
ZAPUSTIT.bat
```

**Или вручную:**
```bash
python -m zavgar_app.main
```

## Использование

1. **Запустите приложение** — откроется главное окно с Dashboard
2. **Добавьте автомобили** — раздел "Автопарк"
3. **Добавьте водителей** — раздел "Водители"
4. **Настройте склад** — раздел "Склад запчастей"
5. **Создайте графики ТО** — раздел "ТО и обслуживание"
6. **Смотрите статистику** — раздел "Отчёты"

## База данных

Приложение использует SQLite (файл `zavgar_data.db` создаётся автоматически).

**Таблицы:**
- `vehicles` — автомобили
- `drivers` — водители
- `parts` — запчасти
- `part_transactions` — транзакции запчастей
- `maintenance_schedules` — графики ТО
- `maintenance_records` — записи ТО

## Разработка

### Добавление нового модуля

1. Создайте модель в `models/`
2. Добавьте CRUD функции в `db.py`
3. Создайте UI страницу в `ui/pages/`
4. Подключите в `main_window.py`

### Переключение темы

```python
from zavgar_app.ui.theme import theme_manager
theme_manager.toggle(QApplication.instance())
```
